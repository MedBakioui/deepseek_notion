Privacy Policy

Last updated: [Date]

1. Données collectées

Cette extension collecte les conversations via l'API DeepSeek pour les enregistrer dans Notion. Aucune donnée personnelle n'est collectée directement par l'extension.

2. Utilisation des données

Les données collectées sont utilisées uniquement pour créer des pages dans votre base de données Notion. Elles ne sont pas stockées ailleurs.

3. Partage des données

Les données sont partagées avec DeepSeek et Notion via leurs APIs respectives. Aucune donnée n'est partagée avec d'autres tiers.

4. Sécurité des données

Nous utilisons des connexions sécurisées (HTTPS) pour protéger les données lors de leur transfert.

5. Droits des utilisateurs

Vous pouvez accéder, modifier ou supprimer vos données directement dans Notion. Pour toute question, contactez-nous à [votre email].

6. Modifications de la politique

Nous pouvons mettre à jour cette politique. Les utilisateurs seront informés des changements importants.
